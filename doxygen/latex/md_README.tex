\doxysection*{Решение квадратного уравнения}

\doxysubsubsection*{Первое задание на летней школе от Деда}\hypertarget{md_README_autotoc_md0}{}\doxysubsection{Описание программы}\label{md_README_autotoc_md0}
Программа решает квадратные уравнения вида ax$^\wedge$2 + bx + c = 0.

Также есть возможность решать линейные уравнения вида kx + b = 0.

Реализованы тестирующая система, система нахождения ошибок, системы ввода и вывода, а также функции для решения квадратного и линейного уравнений.\hypertarget{md_README_autotoc_md1}{}\doxysubsection{Руководство пользователя}\label{md_README_autotoc_md1}
\hypertarget{md_README_autotoc_md2}{}\doxysubsubsection{Запуск программы}\label{md_README_autotoc_md2}

\end{DoxyCode}

\hypertarget{md_README_autotoc_md3}{}\doxysubsubsection{Ответ программы}\label{md_README_autotoc_md3}
Если ошибок нет, программа попросит внести необходимые коэфиценты(три действительных числа a, b, c, которые являются членами уравнения ax$^\wedge$2 + bx + c = 0 в соответственном порядке). Если были введены верные коэфиценты, выводится ответ. Возможные сообщения\+:
\begin{DoxyItemize}
\item Корни, если они есть
\item \char`\"{}\+No roots\char`\"{}, если корней нет
\item \char`\"{}\+Any number\char`\"{}, если все корнем уравнения является любое число
\end{DoxyItemize}\hypertarget{md_README_autotoc_md4}{}\doxysubsection{Функции, константы и структуры}\label{md_README_autotoc_md4}

Функция для еденичного тестирования функции {\bfseries{Solve\+Square}}.

Принимает коэфиценты уравнения в структуре {\bfseries{\mbox{\hyperlink{structCoeffs}{Coeffs}}}}, а также ожидаемые значения корней в структуре {\bfseries{\mbox{\hyperlink{structRoots}{Roots}}}} и колличества корней. Выполняет функцию {\bfseries{Solve\+Square}} на полученных коэффицентах и сравнивает полученный результат с ожидаемым. При совпадении отправляет сообщение ОК зелёным цветов и возвращает константу {\bfseries{Test\+Acepted}} типа {\bfseries{Test\+Feedback}}. При несовпадении отправляет сообщение об ошибке красным цветом с указанием ожидаемых и действительных результатов и возвращает константу {\bfseries{Test\+Failed}} типа {\bfseries{Test\+Feedback}}.\hypertarget{md_README_autotoc_md24}{}\doxyparagraph{Test\+All}\label{md_README_autotoc_md24}

\begin{DoxyItemize}
\item Коэффицент а.
\item Коэффицент b.
\item Коэффицент c.
\item Ожидаемое колличество корней.
\item Ожидаемый меньший корень(0, если корней нет или их бесконечное колличество).
\item Ожидаемый больший корень(0, если корней нет или их бесконечное колличество).
\end{DoxyItemize}

Завпускает функцию {\bfseries{Test\+One}} с полученными данными.

Если тест написан некорректно, то печатает текст, оповещающий об ошибке, и завершает досрочно функцию.

Если все тесты были пройдены, печатается колличество пройденных тестов.

Функция возвращает колличество пройденных тестов.
