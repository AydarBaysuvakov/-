◆ InputCoeffsSTD()

IOError InputCoeffsSTD ( Coeffs *  coeffs_p)

Запрашивает коэффиценты квадратного уравнения вида ax^2 + bx + c = 0.

На первой строке выводится информация о программе На второй строке выводится просьба ввести коэфиценты Далее пользователь должен ввести три действительных числа, коэффиценты a, b, c

Возвращает
OK=0, если пользователь вводит верные данные, при ошибке выводит ERROR=1
